# Prompt Library — AI-Augmented Research Seminar

This folder contains all system prompts used across the six seminar sessions,
saved verbatim as plain-text files. Saving prompts verbatim is a core requirement
of **Competency 3 (Prompt Discipline)** in the Zyphur (2026) framework.

> "Prompts are methodological choices. They must be documented with the same
> rigour as any other methodological decision in the research workflow."
> — Zyphur (2026), *Responsible AI in Academic Research*

## File naming convention

`session{day}_{session}_{function}.txt`

## Usage

Each file contains:
- The verbatim system prompt
- A brief annotation explaining the prompt's purpose and design rationale
- Notes on known failure modes and verification steps
